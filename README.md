# Introduction
This is the core Raito CLI implementation.

# Getting Started

## Install
Using HomeBrew:
```bash
brew install raito-io/tap/cli
```

# Usage
To get an overview of the possibilities, simply use the `--help` flag
```bash
raito --help
```
